# B-Hall プロダクト概要

## サービス名
B-Hall（ビーホール）

## 提供会社
株式会社Backlly

## 一言定義
営業活動以外の中小企業バックオフィス全体を統合し、出来事起点で業務・承認・書類・お金・人・改善活動をつなぐクラウド型バックオフィスOS。

## 対象ユーザー
- 従業員5〜300名規模の中小企業
- 社労士・税理士等の外部専門家
- Backlly社のCS/コンサル担当者

## 核心的価値
機能の多さではなく、バックオフィスのすべてが**一つの企業運営フローとして繋がっていること**。

## 解決する課題
1. バックオフィス業務がツールごとに分断されている
2. 「あれ誰がやるんだっけ」「期限いつだっけ」「この書類どこだっけ」が日常的に発生
3. 経営者が会社の状態を把握するために複数ツールを横断する必要がある
4. 改善活動が属人化し、積み上がらない
5. イベント（入社等）発生後の手続きが漏れる

## 最終的に実現する状態
- 経営者が朝開けば、会社の状態が一目で分かる
- 担当者が開けば、今日やるべきことが整理されている
- バックオフィスで業務の漏れ・遅延・分断が起きない
- 改善活動が可視化され、継続的に積み上がる
- 営業さえ回れば、あとはB-Hallで会社運営基盤が整う
