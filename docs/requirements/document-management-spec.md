# B-Hall 文書管理仕様

## 概要

契約書、規程、証憑、申請添付等すべてのビジネス文書を一元管理する。ファイル基盤（E層）の上にM07の文書管理UIを構築する。

## 文書種別

| 種別 | 例 | 期限管理 | バージョン管理 |
|------|---|---------|-------------|
| 契約書 | NDA、業委契約、雇用契約 | ○ | ○ |
| 規程 | 就業規則、給与規程 | - | ○ |
| 届出書 | 社保届、労基届 | ○ | - |
| 行政文書 | 各種届出控え | - | - |
| 証憑 | 領収書、請求書 | - | - |
| 申請添付 | 申請に添付された書類 | - | - |
| 稟議添付 | 稟議に添付された資料 | - | - |
| マニュアル | 業務手順書 | - | ○ |

## フォルダ構造

テナントごとにルートフォルダを持ち、以下のデフォルト構造を提供:

```
/（ルート）
├── 契約書/
│   ├── NDA/
│   ├── 業務委託/
│   └── 雇用契約/
├── 規程/
├── 届出・行政文書/
├── 証憑/
│   ├── 2026年/
│   │   ├── 01月/
│   │   └── ...
├── 申請添付/
└── その他/
```

カスタムフォルダの追加可能。

## 機能要件

### アップロード
- ドラッグ&ドロップ対応
- 複数ファイル同時アップロード
- 対応形式: PDF, Word, Excel, 画像, テキスト
- ファイルサイズ上限: 50MB/ファイル

### タグ
- 自由タグ付け
- タグでの横断検索

### バージョン管理
- 同一文書の複数バージョン保持
- バージョン間の差分表示は将来対応
- 最新バージョンをデフォルト表示

### 有効期限管理
- 契約書等に有効期限を設定
- 期限接近時に自動通知（→M12）
- 更新タスクの自動生成（→M02）

### 閲覧権限
- 文書単位、フォルダ単位での権限設定
- 閲覧/編集/ダウンロードの制御

### 関連オブジェクト紐付け
- タスク、申請、稟議、取引、従業員との紐付け
- 紐付け先からの文書参照
