# B-Hall 経理・財務仕様

## 概要

経理実務と資金管理をバックオフィス全体と繋ぐ。孤立した会計機能ではなく、申請→承認→支払→会計反映→CF更新の流れを実現する。

## サブモジュール構成

### 取引管理
- 収入・支出・振替の記録
- 勘定科目・補助科目マスタ
- 取引先マスタ
- 証憑（Document）紐付け
- 申請（Application）からの自動取引作成

### 経費精算
- M03ワークフローの経費申請→承認後に精算データ作成
- 領収書（Document）紐付け
- 精算状況（未精算/精算済み/振込済み）
- 支払状況の追跡

### 請求管理
- 請求書の作成・発行
- 入金予定の管理
- 入金消込
- 未収一覧
- 入金遅延アラート（→M12通知）

### 支払管理
- 支払依頼（M03ワークフロー経由）
- 支払予定表
- 支払実行確認
- 未払一覧
- 支払期限アラート（→M12通知）

### 月次管理
- 締めチェックリスト（タスクテンプレート自動生成）
- 試算表確認タスク
- 証憑回収状況
- 残高確認

### CF管理
- 現預金残高のリアルタイム表示
- 月別入出金予測
- 固定支出・臨時支出の区分
- 資金繰りアラート（→M12通知、M01ダッシュボード）

### 財務分析
- 月次収支グラフ
- 費目別分析
- 部門別分析
- 予算対比
- 未収未払サマリー
- CF変動要因分析

## 連携フロー例：経費申請→会計反映

```
従業員が経費申請（M03）
  → 上長承認
  → 経理確認
  → 取引データ自動作成（M08）
  → 支払実行
  → キャッシュフロー更新
  → 月次試算表に反映
```
