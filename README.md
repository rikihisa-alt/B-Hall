# B-Hall

株式会社Backllyが展開するバックオフィス支援システム

## プロジェクト構成

```
B-hall/
├── client/          # フロントエンド (React + TypeScript)
├── server/          # バックエンド (Node.js + Express + TypeScript)
└── docs/            # ドキュメント
```

## セットアップ

### フロントエンド

```bash
cd client
npm install
npm run dev
```

### バックエンド

```bash
cd server
npm install
npm run dev
```

## 技術スタック

- **Frontend**: React, TypeScript, Vite
- **Backend**: Node.js, Express, TypeScript
- **Database**: TBD
